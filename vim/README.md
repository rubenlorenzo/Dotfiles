```

```
## Pathogen
Install Plugins [vim-pathogen](https://github.com/tpope/vim-pathogen) 

```
mkdir -p ~/.vim/autoload ~/.vim/bundle && \
curl -LSso ~/.vim/autoload/pathogen.vim https://tpo.pe/pathogen.vim
```
